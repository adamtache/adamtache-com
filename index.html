Hear4You (Github and Website) Developed online mental health platform (team of 3) focusing on facilitating healthy conversations about mental health and providing dynamic university resources. As the final project for CS190 Adapative Web Development, I was responsible for designing the front-end and the dynamic resources page and worked with Bootstrap, HTML, CSS, Javascript, jQuery, and Ajax. Currently working as a web developer to expand this project.

Conway's Game of Life (Github and Website) I created a client-side simulator of Conway's Game of Life cellular automaton (devised by British mathematician John Conway in 1970) using HTML, CSS, Javascript, and jQuery. I implemented customization options of the initial board dimensions, simulation speed, and the ability for the user to draw individual cells. This is a zero-player game and the automation of cell states is determined after each generation (one step) based on specific rules.

Cache Simulator (Github) I created a simulator of a single-level, set-associative cache with write-back and write-allocate policy using Java object-oriented design. I minimized cache misses by using the LRU replacement scheme based on temporal locality. This was a project for CS250 Computer Architecture.

MIPS Processor I designed with Logisim a fully-functioning 16-bit MIPS-like, word-addressed RISC architecture processor. I implemented a subset of 16 instructions of original MIPS commands, including arithmetic operations, load/store, jumps, and braches as part of CS250 Computer Architecture.
